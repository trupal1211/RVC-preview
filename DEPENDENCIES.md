# RelationshipVista Project Dependencies

This document lists the core technologies and libraries used in the RelationshipVista website.

## Core Framework & Build Tools
- **React 18**: Main frontend library.
- **Vite**: Ultra-fast frontend build tool and development server.
- **TypeScript**: Static typing for enhanced developer experience and code quality.

## UI & Styling
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **Framer Motion**: Production-ready motion library for React (animations).
- **Lucide React**: Beautiful & consistent icon set.
- **Radix UI**: Unstyled, accessible components (Dialog, Dropdown, Accordion, etc.).
- **Sonner**: Opinionated toast component for feedback.
- **Glassmorphism**: Custom CSS utilities for premium, semi-transparent UI elements.

## State Management & Routing
- **React Router Dom**: Standard routing library for React applications.
- **TanStack Query (React Query)**: Powerful asynchronous state management.
- **React Hook Form**: Performant, flexible and extensible forms with easy-to-use validation.

## Security & Reliability
- **Google reCAPTCHA v3**: Automated bot protection for contact forms.
- **Nodemailer**: Node.js library for easy email sending (used in backend).
